Logical Schema (Entity with attributes, primary key, and foreign key)

1.  Passanger
    ---------
    PassengerID  PK
    Gender
    CustomerType
    Age
    TravelType
    Satisfaction
    FlightID FK >-< Flight.FlightID
    BookingID  FK -< OnlineServices.BookingID
    InflightServiceID FK >- InflightServices.InflightServiceID
    AirlineServiceID FK >-AirlineServices.ArilineServiceID    
    
    
2.  Flights
    ------
    FlightID  PK
    Class
    SeatComfort
    FlightDistance    
    Cleanliness
    DepartArriveID FK >- DepartureArrival.DepartArriveID    
    
3.  DepartureArrival
    -----------------
    DepartArriveID  PK
    TimeConvenience
    DepartureDelay
    ArrivalDelay
    
    
4.  InflightServices
    -----------------
    InflightServiceID  PK
    Wifi
    Entertainment
    InflightService
    FoodDrink
    OnboardService
    LegroomService    
    
    
5.  OnlineServices
    ---------------
    BookingID  PK
    EaseOfOnlineBooking
    OnlineBoarding   
    
    
6.  AirlineServices
    ----------------
    AirlineServiceID  PK
    CheckinService
    GateLocation
    BaggageHandling
